Преимущества

  • Адаптивность

    Ваш сайт будет смотреться хорошо как на больших экранах, так и на мобильных устройствах

  • Кроссбраузерность

    Корректное отображение сайта в различных браузерах

  • Семантичность

    Семантическая вёрстка увеличивает доступность информации для посетителей сайта, улучшает поисковую индексацию, облегчает дальнейшую поддержку

Ключевые навыки:

  • Адаптивная кроссбраузерная верстка с использованием HTML5 и CSS3
  • Базовые знания JavaScript и JQuery
  • Работа с Photoshop (нарезка макетов)
  • Использование методологии БЭМ в именовании классов
  • Работа с Git через командную строку
  • Использование SASS
